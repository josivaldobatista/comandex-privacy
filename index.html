Política de Privacidade

Esta Política de Privacidade descreve como o Comandex coleta, utiliza e protege as informações fornecidas pelos usuários da aplicação.

1. Sobre o Comandex

O Comandex é um sistema de gestão para restaurantes, lanchonetes e estabelecimentos similares, auxiliando no controle de produtos, pedidos, mesas, pagamentos e relatórios.

2. Informações coletadas

Coletamos informações necessárias para autenticação e funcionamento da plataforma, como e-mail, dados da empresa, produtos cadastrados, pedidos, mesas e usuários autorizados.

3. Uso das informações

Os dados são utilizados exclusivamente para operação do sistema, autenticação dos usuários, organização dos pedidos e exibição das informações necessárias dentro da aplicação.

4. Compartilhamento de dados

Não comercializamos dados pessoais com terceiros. As informações são utilizadas apenas para viabilizar o funcionamento do Comandex.

5. Armazenamento e segurança

Os dados são armazenados em infraestrutura utilizada pela aplicação, com medidas razoáveis de segurança para proteger as informações contra acessos não autorizados.

6. Solicitação de remoção de dados

O usuário pode solicitar informações, correções ou exclusão de dados entrando em contato pelo e-mail: josivaldo.fb@gmail.com.

7. Alterações nesta política

Esta política pode ser atualizada periodicamente para refletir melhorias no aplicativo ou mudanças legais e operacionais.

Última atualização: Maio de 2026.
